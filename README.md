## Featured projects

### 🏠 HostelOS &nbsp;<sub><i>flagship</i></sub>

> The operating system for your hostel. A multi-tenant SaaS that replaces scattered spreadsheets and WhatsApp threads with a single source of truth — built for Indian PG / hostel managers running 100–1,000 residents per property.

<p>
  <a href="https://hostel-os-zeta.vercel.app/">
    <img src="https://img.shields.io/badge/Live%20Demo-→-22c55e?style=for-the-badge&labelColor=0a0a0a" alt="Live Demo"/>
  </a>
  &nbsp;
  <img src="https://img.shields.io/badge/Status-Production-3FCF8E?style=for-the-badge&labelColor=0a0a0a" alt="Status: Production"/>
</p>

- 🏢 **Multi-tenant** — one deployment serves many hostels, isolated by Postgres row-level security
- 💸 **Billing engine** — monthly invoices auto-generated via `pg_cron`, late fees, partial payments
- 📱 **UPI integration** — one-tap deep links in WhatsApp + email reminders, public claim form for UTR
- 🔐 **Self-service auth** — OTP signup, rate-limited password reset, role-based access (owner/manager/staff/resident)
- 🧾 **DPDP-compliant** — auto-purges resident ID proofs 6 months after checkout, EXIF stripped on upload
- 📊 **Reports** — billing vs collection trends, payment-mix breakdown, occupancy %, ticket SLA

<sub><b>Stack:</b> Next.js 14 · TypeScript · Supabase (Postgres + RLS + Edge Functions) · Tailwind · Sentry · Vercel</sub>
